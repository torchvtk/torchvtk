Transformations¶

For Transformations it is necessary to call the Subclasses of the DictTransform Super Class. An example is presented in the following code snipped.

    tfms = NoiseDictTransform(noise_variance=(0.01, 0.1), device="cpu", apply_on=["vol"])
    tmp = tfms(file)
class torchvtk.augmentation.DictTransform(device='cpu', apply_on=['vol', 'mask'], dtype=torch.float32)¶

Bases: object

Super Class for the Transforms.

__init__(device='cpu', apply_on=['vol', 'mask'], dtype=torch.float32)¶
Parameters

  • device – The torch Code for the device on which the transformation should be executed. Possiblities are [“cpu”, “cuda”].

  • apply_on – The keys of the volumes that should be transformed.

  • dtype – The torch type in which the data are. Possibilities are [torch.float16, torch.float32].

Transformation Method, must be overwritten by every SubClass.
